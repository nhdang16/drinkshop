# Backend Setup Guide

This guide explains how to set up and run the backend project locally.

## 1. Install Required Tools

### 1.1 Install IntelliJ IDEA
Download IntelliJ IDEA from the official website:

https://www.jetbrains.com/idea/download/

Recommended version: **IntelliJ IDEA Community Edition** (free).

After downloading:
1. Install IntelliJ.
2. Open IntelliJ.
3. Choose **Open Project** and select the backend project folder.

---

### 1.2 Install PostgreSQL

Download PostgreSQL from the official website:

https://www.postgresql.org/download/

Steps:
1. Download PostgreSQL installer.
2. Install PostgreSQL.
3. During installation:
   - Set **1** for the password
   - Default port: **5432**

PostgreSQL will also install **pgAdmin** for database management.

---

## 2. Create Database in pgAdmin

1. Open **pgAdmin**.
2. Connect to your PostgreSQL server.
3. Right click **Databases**.
4. Select **Create → Database**.
5. Database name: **drinkshop**
6. Click **Save**

## 3. Run the Application by IntelliJ Idea
